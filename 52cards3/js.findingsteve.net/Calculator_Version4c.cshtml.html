Simple Calculator - Version 4c

Actual Output

Features
  1. Display temporary result while entering number (new)
  2. Percentage % key (new)
  3. Back key (new)
  4. Added CSS
  5. Keep appending string calculation like in Android stock calculator. Example, 1+2*2
  6. Order of calculation is BODMAS. Example, 1+2*2=5 (not 6). For Window calculator, result is 6.
  7. Accept number key from 1-9
  8. Accept decimal
  9. Accept zero
Code
  • Main function - One addEventListener for the 0-9 keys, decimal key and all four operator (+ - * /) keys are dynamically generated using foreach loop
  • Total small functions: 5
  • Total anonymous functions: 2
  • Some Javascript built-in function used: eval(), split(), includes()
  • Total lines of code: 83 (excluding comment)
  • Total Passed Test Cases: 15
  • Main Design Ideas: Use boolean to check key type and last number string.
  • CSS Grid Layout
  • More comment in the code.
