Features
  1. Added CSS (new)
  2. Keep appending string calculation like in Android stock calculator. Example, 1+2*2
  3. Order of calculation is BODMAS. Example, 1+2*2=5 (not 6). For Window calculator, result is 6.
  4. Accept number key from 1-9
  5. Clear the expression string when equal "=" button is pressed
  6. Accept decimal
  7. Accept zero
